# SEO Ecommerce Sales Assistant - Specification

## Vision
Build a production-ready SEO & Sales life cycle personalization and CRO (Conversion Rate Optimization) React application. The goal is to increase conversion and revenue through intelligent, data-driven user experiences.

## Core Value Propositions
1.  **SEO First**: Technical SEO built-in (SSR/Pre-rendering, Metadata, Structured Data).
2.  **Personalization**: Dynamic content based on user behavior and sales lifecycle stage.
3.  **Semantic Search**: AI-powered search using Azure OpenAI and Azure AI Search to understand user intent.
4.  **CRO**: High-converting UI patterns, A/B testing capabilities, and analytics integration.

## Technology Stack
*   **Frontend**: React (Vite) + TypeScript
*   **Styling**: TailwindCSS
*   **State Management**: Zustand
*   **Routing**: React Router v6
*   **Cloud Infrastructure**: Azure (Static Web Apps, AI Search, OpenAI)
*   **AI/ML**: Azure OpenAI (Embeddings), Azure AI Search (Vector Store)

## Deployment
*   **Platform**: Azure Static Web Apps
*   **CI/CD**: GitHub Actions
